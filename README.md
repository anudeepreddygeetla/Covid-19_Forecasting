# Covid-19_Forecasting
Time series forecasting of COVID-19  using Facebook Prophet.
# Death Rate Forecasting using Prophet

## Dataset
- Time range: February 2020 to August 2020
- Records: (49068 rows × 10 columns)

## Model Details
- Algorithm: Facebook Prophet
- Training period: Feb - Aug 2020
- Forecast horizon: 30 days ahead

## Results
(I have done 2 fecastings)
* The Covid-Cases for future 7 days and
* The Covid-Deaths for future 30 days
